# nix-pres-draft

A short [Slidev](https://sli.dev) deck introducing Nix.

```bash
npm install
npm run dev      # live preview at http://localhost:3030
npm run build    # static build into dist/
npm run export   # export to PDF
```

Tooling: Prettier (`npm run fmt`), markdownlint-cli2 (`npm run lint`), knip (`npm run deadcode`).
